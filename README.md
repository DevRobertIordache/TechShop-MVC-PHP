1. üîΩ [Download the project as ZIP](https://github.com/YOUR_USERNAME/tech-shop-ecommerce/archive/refs/heads/main.zip)

4. Visit [http://localhost/phpmyadmin](http://localhost/phpmyadmin)
5. Create a database called `ecommerce`
